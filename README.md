
Project Overview

This project analyzes synthetic healthcare data using Python to identify risk levels based on patient health parameters like blood pressure,
sugar level, cholesterol, and heart rate.

Objectives

Generate synthetic patient data
Analyze heart rate and classify status
Identify high-risk and low-risk patients
Group patients by age
Perform data analysis and visualization

Tools Used

Python, Pandas, NumPy, Matplotlib

Methodology

1. Create dataset using NumPy
2. Process data using Pandas
3. Apply conditions for risk classification
4. Visualize data using charts
 
Output & Visualization

The project generates multiple visualizations such as bar charts, pie charts, histograms, scatter plots, box plots, and heatmaps to understand data patterns.

Conclusion

This project helps in understanding patient health risks and provides insights using data analysis techniques.
